# Neural Network from Scratch (NumPy + PyTorch)

This project demonstrates the implementation of a fully connected neural network using two approaches:

1. A **from-scratch implementation using NumPy** to understand the mathematical foundations of neural networks
2. A **PyTorch-based implementation** to replicate the same architecture using an industry-standard deep learning framework

The objective of this project is to build a strong conceptual and practical understanding of forward propagation, backpropagation, gradient descent optimization, and training workflows.

---

## Features

- Implemented a neural network from scratch using NumPy
- Manually coded forward propagation and backpropagation
- Implemented gradient descent for parameter updates
- Built configurable training loops
- Reimplemented the same architecture using PyTorch
- Used automatic differentiation with PyTorch autograd
- Evaluated performance using loss and accuracy metrics

---

## Tech Stack

- Python
- NumPy
- PyTorch
- Matplotlib (for visualization)

---

## Learning Outcomes

Through this project, I developed practical understanding of:

- neural network architecture design
- forward and backward propagation
- gradient-based optimization
- loss function behavior during training
- hyperparameter tuning
- differences between manual and framework-based implementations

---


## Results

Both implementations were evaluated using training loss and accuracy metrics. The project demonstrates equivalent learning behavior between manual gradient-based training and framework-assisted training.
